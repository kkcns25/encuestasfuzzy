Un poco sobre este proyecto

Éste proyecto comenzo para dar cabida a la realización de encuestas, en las cuales se muestre el grado de aceptación a al título de cada cuestión. El modelo clásico es el denominado modelo Likert, el cual da la posibilidad de seleccionar el grado de aceptación, pero el cual siempre es un número fijo, dando, comunmente, entre 3 y 7 posibles grado de aceptación, como vemos en la siguiente imagen;



Ante este panorama,nació esta la posibilidad de realizar encuestas en la que la posible respuesta tenga total grado de libertad, para mostar la aceptación de cada pregunta, este tipo de encuestas se llaman Fuzzy Likert.
